# SQL Data Analysis Project

## Objective

Perform data analysis on an E-Commerce dataset using SQL and extract meaningful business insights.

## Tools Used

* MySQL Workbench
* SQL

## Dataset

* Orders.csv
* Details.csv

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* Aggregate Functions (SUM, AVG, MAX, MIN)
* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* Subqueries
* Views

## Analysis Performed

* Total Sales Analysis
* Profit Analysis
* Sales by Category
* Sales by State
* Payment Mode Analysis
* Customer Order Analysis

## Files Included

* Dataset files
* SQL query file
* Output screenshots
* README documentation

## Outcome

This project demonstrates the use of SQL for data cleaning, data retrieval, aggregation, joins, and business analysis on an e-commerce dataset.
